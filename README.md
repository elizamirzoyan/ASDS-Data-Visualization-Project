# Exploratory Data Analysis Project

This project contains EDA on three beauty campaign datasets:

- Nykaa
- Purplle
- Tira

## Project Structure

- `data/` contains the CSV files
- `eda_analysis.ipynb` contains the notebook
- `environment.yml` contains the conda environment
- `.gitignore` ignores unnecessary files

## How to Run

```bash
conda env create -f environment.yml
conda activate eda-env
jupyter notebook
